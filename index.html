Shiny App: First Take Away

In this app, there is some analysis about the Craiglist data set, the world’s largest collection of used vehicles for sale. We will find the following pages:

  1. Craiglist's vehicles: There are an introduction of the problem, how we are going to deal with it and also, we describe the variables we are going to use.

  2. Data description: There are a left panel where we can select any of the variables we have defined before and the data frame description, where we can show a different number of entries, search any value or sort the variables.

  3. Descriptive analysis: In the following panel, we are able to plot two graphics, an histogram and a barplot, to see the relationship between our target variable and the rest of predictors.

  4. Distribution of the vehicles: In this page, we can observe two interactive maps of the USA: on the left, there is the location of the vehicles, and on the right, the quantity of vehicles on sale.

  5. Statistical models: In this last page, we can perform some statistical and machine learning models, selecting the partition of the data that goes in the training set, the method of resampling and the model, which can be generalized linear model (glm), k nearest neighbors (knn), random forest (rf) or neural network (nnet).
